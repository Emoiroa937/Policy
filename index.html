We are fully committed to protecting your privacy. This Privacy Policy outlines how the "Hỗ trợ nghiệp vụ cấp xã" extension handles and protects your data while operating on authorized official document management systems.

1. Data We Process (Data Collection)

In accordance with our declarations on the Chrome Web Store, our extension accesses and temporarily processes the following data locally on your device:

  • User Activity: We monitor specific mouse click events (e.g., clicking on action buttons such as #ChuyenDongThoi, #ChuyenXuLyUBND). This is strictly necessary to trigger consecutive automated workflow actions.
  • Website Content: We read structural data from the DOM, specifically document identification numbers (such as data-tphcm or data-fly-ids attributes), to transfer task information accurately between authorized administrative web pages.

2. Permissions Justification

  • Storage Permission: The chrome.storage.local API is used exclusively to store temporary, session-based commands (including the action type, task ID, and timestamp). This allows the extension to maintain automation continuity across page reloads and tab navigations.
  • Host Permissions: The extension requires access to specific domains (phuongbinhtay.vn, qlvb.tphcm.gov.vn, and dhtn.dcs.vn). The extension strictly operates only on these authorized domains and does not interact with or read data from any other websites.

3. Data Protection and Privacy Commitments

  • 100% Local Processing: All data involving user activity, document content, and temporary commands are processed entirely offline within your local browser environment.
  • No Data Transmission: We do not operate external servers. The extension never transmits, shares, sells, or uploads any personal or administrative data to the Internet or third parties.
  • Automated Data Deletion: Any command data temporarily saved in the local storage is automatically erased either immediately after the automated task completes or within a maximum of 120 seconds.

4. Contact Information

If you have any questions or concerns regarding this Privacy Policy, please contact the developer via the email address provided on the Chrome Web Store Developer Dashboard.
